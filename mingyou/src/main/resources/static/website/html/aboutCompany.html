公司介绍

2016.12.06

2016年12月1日，中国保监会批复同意招商局仁和人寿保险股份有限公司的筹建申请。此次复牌的仁和人寿源于招商局1875年创办的中国第一家真正意义上的民族保险公司。“仁和保险”成功复牌，光复了中国民族保险品牌，并将中国民族保险业的历史提前了近60年。

1875.12.28

1875年12月28日，由轮船招商总局筹设的仁和保险前身——保险招商局，在上海宣告成立。百余年来，它从保险招商局到仁和保险公司，再与济和保险重组合并为仁济和保险公司，历经晚清与民国，融入到新中国的保险事业之中，成为中国近代民族保险业的先驱。

招商局仁和人寿保险股份有限公司注册资本为50亿元人民币，由招商局集团作为主发起人，共同发起股东包括中国移动通信集团公司、中国民航信息网络股份有限公司、深圳市投资控股有限公司、深圳市前海金融控股有限公司、深圳市卓越创业投资有限责任公司、深圳光汇石油集团股份有限公司和亿赞普（北京）科技有限公司。

招商局仁和人寿保险股份有限公司将践行“保险姓保”的理念，坚持质量、规模、效益动态均衡发展，

充分整合股东资源，创新商业模式，培育市场化、专业化、差异化的竞争优势，满足不同客户全方面多层次的保险需求,

立足深圳，面向全国，塑造百年品牌，致力于成为国内领先，独具特色和品牌价值的综合保险服务商。
